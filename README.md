SunshineV
===============

![show](./SunshineV.gif)  

## Requirements

- iOS 8.0+
- Xcode 7.0

## Usage

```swift

let sV = SunshineV(frame: CGRectMake(0, 0, 200, 200))
sV.text = "Sunshine\nMagic"

view.addSubview(sV)
sV.run()

```
